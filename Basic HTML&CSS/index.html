Section #1: paragraph and anchor elements

This is a paragraph element. Paragraphs are block elements, which means that by default they take up 100% of the screen width.

Inside paragraphs, we can have links, which in HTML are called "anchors". Anchors are inline elements, which means that they don't alter the flow of the document (they stay inline with the text). Through the attribute "href", anchors allow us to reference other sites or different sections of the same page.
